# 63岁生日视频脚本（轻松版）

> 不出镜、不煽情、像家庭内部看的那种

---

## 整体感觉

就像你平时跟他们说话一样，不用"岁月如梭"那种词。  
核心就一句：**"我虽然不搞这些，但今天替你们搞一个，因为今天是你们俩的日子。"**

时长：1分30秒 - 2分钟，够了。

---

## 第一段：开门见山（0:00-0:15）

**画面**：随便一张他俩最近的合影，或者家里的某个角落，或者你今天随手拍的一张

**字幕**：

```
平时我不爱拍照
也不发抖音
但今天不一样

今天你们俩过生日
我也来做一次你们喜欢的事
```

**音乐**：轻快的纯音乐进，音量小

---

## 第二段：展示他俩（0:15-1:00）

**画面**：照片轮播，不用按时间顺序，按"主题"分类放

- **主题1：徒步**（3-4张）
- **主题2：拍照**（2-3张，最好有他们举着相机的）
- **主题3：母亲的抖音时刻**（可以手机录屏她抖音页面，或者她对着手机拍的侧影）
- **主题4：舅舅的沉浸式阅读**（如果有他躺着看手机的/看书的，没有就跳过）

**字幕**（配在画面下方，像聊天一样）：

```
一个爱拍抖音
一个爱看小说
俩人都爱徒步
还非要住在一起
```

然后可以空一行，出一句：

```
行吧，反正你们开心就行
```

---

## 第三段：核心（1:00-1:20）

**画面**：一张他们俩在一起的背影/侧面，或者并排坐的，不要正脸大头照，随意一点

**字幕**：

```
63岁了
我没别的可说
就觉得你们现在这样
挺好的

继续
```

---

## 第四段：收尾（1:20-1:50）

**画面**：同一张照片慢慢定格，或者放一张今天拍的（蛋糕、饭菜、家里环境都行）

**字幕**：

```
生日快乐
妈
舅

—— 做视频这活儿确实挺累的
    你们以后多发点，我继续白嫖着看
```

**音乐**：淡出，结束

---

## 音乐建议

不要抒情的。轻快、日常感：

- **《New Boy》朴树**（原版或轻音乐版）——有"新的一天"的感觉，不肉麻
- **《米店》张玮玮**（民谣吉他版）——温暖但克制
- **《菊次郎的夏天》**——轻松，缺点是听太多可能腻
- **或者直接去剪映音乐库搜"轻快""日常""vlog"**，挑一个顺耳的，音量压到20%

---

## 剪映操作（极简）

1. 建项目，**9:16竖屏**
2. 照片拖进去，每张3-4秒
3. 加字幕：白字，放画面下方，字号别太大
4. 加音乐，音量调小
5. 照片之间用"叠化"或直接硬切，别搞太花哨的转场
6. 导出

---

## 素材不够怎么办

- 没有舅舅看小说的照片？拍一张今天他看手机的背影，或者干脆跳过这个主题
- 没有徒步照片？去母亲抖音录屏几个她的视频片段，当素材用
- 今天随手拍几张：他们在厨房、在阳台、在看电视、蛋糕、饭菜——这些比老照片更真实

---

## 关键提醒

- **不要加滤镜**，原色最好
- **不要写"感恩""养育之恩"**，你们关系好，这些话平时肯定说过，视频里不必重复
- **越像随手做的，越有诚意**——你本来就不是专业搞这个的，他们太清楚了
- **最后那句"白嫖"很重要**，它把你拉回到你平时的样子，而不是"今天突然变了一个人"
