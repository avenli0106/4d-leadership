# AI_Study 项目指南

## 项目背景

这是一个个人学习资料整理项目，主要用于管理和整理运动科学/健身相关的课程音频、笔记和学习资料。

## 当前内容

- **课程音频**：`L87_运动解剖.mp3` — 运动免疫学课程（约1小时）
- **转录文本**：`transcription.txt` — 音频完整文字稿
- **课程笔记**：`course_notes.md` — 结构化整理笔记

## 工作规范

### 文件命名
- 音频文件：`L{编号}_{主题}.mp3`
- 转录文本：`transcription.txt`（每次新课程覆盖或重命名存档）
- 课程笔记：`course_notes.md` 或 `{主题}_notes.md`

### 音频处理流程
1. 将课程音频放入项目根目录
2. 运行 Whisper 转录（small 模型，中文）
3. 读取转录文本，提取核心知识点
4. 生成结构化 Markdown 笔记
5. 笔记包含：知识框架、重点表格、实用建议、后续学习建议

### 笔记格式标准
```markdown
# {主题}课程笔记（L{编号}）

> **课程时长**：约X小时  
> **核心主题**：XXX  

---

## 一、{主要章节}
（分点阐述，使用比喻帮助理解）

## 二、{主要章节}
（善用表格对比）

## 三、实用建议
（可直接应用于训练/生活的要点）

---

## 接下来你可以怎么用这些内容？
（给出 3-5 条具体的后续行动建议）
```

### 技术栈
- **音频转录**：Python + OpenAI Whisper (small模型)
- **依赖工具**：ffmpeg、Python 3.11
- **笔记格式**：Markdown

## 常用命令

```powershell
# 转录音频（已安装好环境）
python -m whisper audio.mp3 --model small --language Chinese --output_format txt

# 生成笔记
python generate_notes.py
```

## 学习目标

1. 建立完整的运动科学知识库
2. 将课程知识转化为可执行的健身/训练方案
3. 通过笔记和卡片实现长期记忆和快速复习
